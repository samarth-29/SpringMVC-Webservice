# SpringMVC-webservice

### Requirement
 - JDK
 - Spring Tool Suite
 - MySql Server
 
### Compile your java code by executing following command:
 - mvn clean install 
 
### Run spring boot application:
 - mvn spring-boot:run OR java -jar mobile-app-webservice-0.0.1-SNAPSHOT.jar
